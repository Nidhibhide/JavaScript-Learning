2.what is currying ?
-Currying can be implemented using Closures or the bind().
-It improves code reusability, readability, and maintainability.
-Currying in JavaScript is a technique that converts a function with multiple arguments into a series of functions that take one argument at a time.

3.what is async programming vs sync programming ?
1]Sync :
-Executes code line by line in sequence.
-It is blocking, so other code waits until the current task completes.

2]Async :
-Allows tasks to run without waiting for previous tasks to finish.
-It is non-blocking, so other code can continue executing while a task is in progress.
-Used for operations like API calls, file handling, timers, and database requests.
